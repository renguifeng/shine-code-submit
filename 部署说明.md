shine-code-submit 新机器部署说明

前置：机器上要有 Node.js（`npx` 要用）。没有的话去 https://nodejs.org 装一个。

1，安装（一条命令）

npx shine-code-submit install

> 国内 npm 若默认走镜像（npmmirror），新版同步有延迟；拉不到最新版时加 `--registry=https://registry.npmjs.org/` 指官方源。

这条命令会自动装 Bun、部署 plugin、注册并启用、拉起 daemon、打印 Dashboard 链接。

2，装完退出 claude，到一个项目目录重新打开 claude、开新会话。正常会弹出网页（或会话顶部打印 Dashboard 链接），显示当前项目的记录。

如果开新会话没反应：在终端分别跑 `bun --version` 和 `node --version`，确认都在 PATH 里（命令本身会自动拉起 daemon，拉起失败则静默跳过，退出码恒 0、绝不阻断 claude）；确认后重开 claude 再试。

3，卸载

npx shine-code-submit uninstall

（不要用 `sudo` —— sudo 没有 nvm 的 PATH，会 `npx: command not found`。）
